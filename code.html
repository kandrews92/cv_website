  • Shell/Bash

    Script mk is a bash script that can be used to run and re-run the pdflatex command and the bibtex. Place file in desired path (preferably with the .tex file) and then use $ ./mk to run the script.
        Download: mk
    Script clean is a bash script that can used to clean out the contents of a directory or directories. Works similar to the command make clean, but is currently configured for use with LaTeX files and removes the *.log,*.bbl, *.out, etc... files, useful when using a repository to save space. Place in desired top directory and then use $ ./clean to run the script
        Download: clean
    Script backup that allows user to backup a specfic directory or path to a server. To use you may need to become root (i.e. $ sudo su) and then run $ chmod +x (script name) (to make it executable) and then move the script to /bin/ so that it can be used as a command. To be used for specific use, edit the username to fit your specific username on the external server, edit the servername and edit the password accordingly. Optional edits can be made to change the source directory target to reflect the folder(s) to be backed-up, adding to crontab by running $ crontab -e in the /home/user/ path and set the approriate time that you would like to run it. (Script created in collaboration with Kevin Andrews)
        Download: backup

  • Fortran

    Below is polymer.tar. This contains a Makefile that resolves dependencies for the file Poly3D.f95. To compile plplot is needed (plplot needs to be downloaded first). plplot allows for the visualization of a simple 3D polymer. The polymer is created subject to a Lennard-Jones potential \begin{equation} V_{LJ}(r) = 4\epsilon\left[\left(\frac{\sigma}{r}\right)^{12}-\left(\frac{\sigma}{r}\right)^{6}\right], \end{equation} with the bond angle chosen via a random angle. Then the polymer chain is built through a Monte Carlo accept/reject method using the Partition function of the system. Making the file and running the executable will show the visualization of a 3D polymer chain. This code can be expanded on, if desired, by adding a bonding potential term and a Coulomb potential term into the calculations. (Code created in collaboration with Daniel Woolridge)
        Download:: polymer.tar

  • LaTeX

    Below is beamer_example.tar. This contains presentation.tex and presentation.pdf. presentation.tex uses the beamer LaTeX class (documentation) to generate a PowerPoint style presentation. This allows for direct use of the LaTeX math environment and incorporation of figures (and much more). This example file is from a talk I gave on a simple model for the mass limits of a neutron star at Michigan State University. The TeX file can be used and edited as a template for future use.
        Download:: beamer_example.tar
        Download:: presentation.tex
        Download:: presentation.pdf
