Some possibly useful snippets of code that I've developed and accumulated over the years. Feel free to use and edit as you see fit. If you notice any mistakes or bugs please contact me.

  • Shell/Bash Scripts

    Script mk is a bash script that can be used to run and re-run the pdflatex command and the bibtex. Place file in desired path (preferably with the .tex file) and then use $ ./mk to run the script.
        Download: mk
    Script clean is a bash script that can used to clean out the contents of a directory or directories. Works similar to the command make clean, but is currently configured for use with LaTeX files and removes the *.log,*.bbl, *.out, etc... files, useful when using a repository to save space. Place in desired top directory and then use $ ./clean to run the script
        Download: clean
    Script backup that allows user to backup a specfic directory or path to a server. To use you may need to become root (i.e. $ sudo su) and then run $ chmod +x (script name) (to make it executable) and then move the script to /bin/ so that it can be used as a command. To be used for specific use, edit the username to fit your specific username on the external server, edit the servername and edit the password accordingly. Optional edits can be made to change the source directory target to reflect the folder(s) to be backed-up, adding to crontab by running crontab -e in the /home/user/ path and set the approriate time that you would like to run it. (Partial credit to Kevin Andrews for this script)
        Download: backup

  • Python

    Class inexact_float.py, used to compare to floating point variables, checks for equality within a user-defined threshold value
        Download: inexact_float.py
    Program excel_extract.py extracts data from an Excel spread sheet and puts the data into an array. Program requires package xlrd. To get package, run command sudo pip install xlrd
        Download:: excel_extract.py
    Program file_list.py, searches current path where program is located and lists files with the specified file extension (i.e. .xls or .xlsx). Allows user to choose from list of files and returns file number and name
        Download:: file_list.py

  • C++

    Below is .tar that contains both files Makefile and main.cpp. The Makefile is a generic one that links the plplot package correctly (plplot needs to be downloaded first) and links the appropriate dependencies and creates a file main.exe which can be run via the command $ ./main.exe. The file main.cpp runs an example of a plotting script of a function that is defined within the program. Output can be chosen via user-input to various formats.
        Download:: plplot_example.tar
        Download:: Makefile
        Download:: main.cpp
